Приложение FitnessApp.
На текущий момент - это консольное приложение для отслеживания тренировок.
В данном приложении можно создать программу тренировок из 8 доступных силовых упражнений.
После этого можно начать тренировку и записать повторение на каждый подход.
После завершения тренировки есть возможность посмотреть последнюю законченную тренировку.
